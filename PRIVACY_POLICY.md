# Privacy Policy for ProRise AI - LinkedIn Assistant

**Last Updated:** November 11, 2025  
**Effective Date:** November 11, 2025

## Introduction

ProRise AI - LinkedIn Assistant ("we", "our", or "the Extension") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our Chrome extension.

## Information We Collect

### 1. User-Provided Information
- **OpenAI API Key**: You provide your own OpenAI API key which is stored securely in Chrome's sync storage
- **Extension Settings**: Your preferences for tone, comment length, and feature toggles
- **Keyword Alerts**: Keywords you configure for monitoring LinkedIn posts
- **WhatsApp Number** (Optional): If you enable WhatsApp notifications

### 2. Usage Statistics
- Number of posts generated
- Number of comments assisted
- Number of replies assisted
- Number of messages replied to
- Number of text rewrites
- These statistics are stored **locally on your device only**

### 3. LinkedIn Content (Temporary Processing)
- Post content you choose to enhance
- Comments you write or want suggestions for
- Messages you want AI assistance with
- **Important**: This content is processed temporarily and sent to OpenAI's API for generation. We do not store or transmit this data to our servers.

## How We Use Your Information

### Primary Uses:
1. **AI Content Generation**: Your LinkedIn content is sent to OpenAI's API (using your API key) to generate suggestions, rewrites, and enhancements
2. **Settings Synchronization**: Settings are synced across your Chrome browsers using Chrome's built-in sync storage
3. **Usage Tracking**: Statistics are stored locally to show your usage patterns
4. **Keyword Monitoring**: To alert you when specified keywords appear in LinkedIn posts

### We Do NOT:
- ❌ Store your LinkedIn content on our servers
- ❌ Share your data with third parties (except OpenAI for processing)
- ❌ Collect personally identifiable information
- ❌ Track your browsing activity outside LinkedIn
- ❌ Sell or monetize your data

## Third-Party Services

### OpenAI API
- **Purpose**: AI content generation and text processing
- **Data Shared**: Post content, comments, messages you choose to enhance
- **Your Control**: You provide your own API key and control what content is processed
- **Their Privacy Policy**: https://openai.com/policies/privacy-policy

### WhatsApp (Optional Feature)
- **Purpose**: Send keyword alerts via WhatsApp
- **Data Shared**: Alert notifications with matched keywords
- **Your Control**: This feature is optional and disabled by default

## Data Storage

### Local Storage (Chrome Extension)
- Usage statistics
- Extension preferences
- Stored **only on your device**

### Chrome Sync Storage
- OpenAI API key (encrypted by Chrome)
- Extension settings
- Synchronized across your Chrome browsers if Chrome sync is enabled

### Our Servers
- **None**: We do not operate any servers or store any of your data

## Data Security

### Security Measures:
1. **API Key Protection**: Your OpenAI API key is stored in Chrome's secure sync storage
2. **No Data Transmission**: We don't transmit your data to our servers
3. **HTTPS Only**: All API requests use secure HTTPS connections
4. **Minimal Permissions**: We request only necessary Chrome permissions

### Your Responsibilities:
- Keep your OpenAI API key confidential
- Use a unique API key for this extension
- Monitor your OpenAI usage and billing
- Revoke API key if you uninstall the extension

## Your Rights and Choices

### You Have the Right To:
1. **Access**: View all your stored settings in the extension popup
2. **Modify**: Change any settings or preferences at any time
3. **Delete**: Uninstall the extension to remove all stored data
4. **Opt-Out**: Disable any feature you don't want to use

### How to Exercise Your Rights:
- **View Data**: Click the extension icon → Settings tab
- **Delete Data**: Uninstall the extension from chrome://extensions/
- **Clear Statistics**: The extension will provide a clear statistics option in future updates

## Permissions Explained

### Why We Need Each Permission:

| Permission | Purpose |
|------------|---------|
| `storage` | Store your settings and statistics locally |
| `activeTab` | Access LinkedIn pages to provide AI assistance |
| `scripting` | Inject AI helper buttons into LinkedIn interface |
| `notifications` | Alert you about keyword matches |
| `tabs` | Monitor LinkedIn tabs for keyword alerts |
| `alarms` | Schedule periodic keyword scanning |
| `clipboardWrite` | Copy generated content to clipboard |

### Host Permissions:
- `https://www.linkedin.com/*` - Required to provide AI assistance on LinkedIn
- `https://api.openai.com/*` - Required to communicate with OpenAI's API
